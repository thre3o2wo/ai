Ch. 4 CSS 선택자

CSS 선택자는 "누구(Which element)에게" 스타일을 적용할지 지정하는 '규칙'. 

---

1. 기본 선택자 (Basic Selectors)

가장 기본이 되는 3가지 선택 방식.

- 태그 선택자 (Tag):
HTML 태그 이름(h1, li, p 등)을 직접 지정한다.
- 전체 선택자 (*):
페이지의 모든 요소를 선택. 주로 margin:0; padding:0; 처럼 페이지 스타일을 '초기화'할 때 사용.
- ID 선택자 (#):#header처럼 # 뒤에 고유한 id 이름을 지정한다. ID는 페이지 내에서 단 한 번만 사용해야 한다.
- Class 선택자 (.):.center처럼 . 뒤에 클래스 이름을 지정한다. Class는 여러 요소에 반복해서 사용할 수 있다.

---

2. 관계 선택자 (Combinators)

요소들 간의 관계(부모-자식-형제)를 기반으로 선택한다.

- 후손 선택자 (   띄어쓰기):#nav ul li처럼 띄어쓰기로 구분한다. #nav *안에 있는* 모든 li를 선택.
- 자손 선택자 (>):#header > div처럼 > 기호를 사용합니다. #header의 *바로 아래 자식*인 div만 선택한다 (손자는 제외).
- 동위(형제) 선택자:
    - h3 + div (인접 형제): h3 태그 *바로 다음에 오는* div 하나만 선택.
    - h3 ~ div (일반 형제): h3 태그 *다음에 오는* 모든 div 형제들을 선택.

---

3. 속성 및 가상 선택자 (Attributes & Pseudo-selectors)

요소의 속성이나 특정 상태/상황을 기반으로 선택한다.

- 속성 선택자 ([]):input 태그처럼 같은 태그라도 속성에 따라 다르게 지정할 수 있다.
    - input[type='password']: type이 password인 요소.
    - input[type^='te']: type이 'te'로 시작하는 요소 (text, tel 등).
    - a[href $= "docx"]: href가 'docx'로 끝나는 요소.
- 반응 선택자 (User Action):
사용자의 행동에 따라 반응한다.
    - :hover: 마우스를 요소 위에 올렸을 때.
- 상태 선택자 (State):
요소의 현재 상태를 나타냄.
    - :focus: input 창을 클릭(포커스)했을 때.
    - :disabled / :enabled: 요소가 비활성화/활성화되었을 때.
    - :read-only: 읽기 전용일 때.
- 구조 선택자 (Structural): 문서 구조상 순서를 기반으로 선택. 10_구조선택자.html에서 연습함.
    - :first-child: 형제 중 첫 번째 요소.
    - :last-child: 형제 중 마지막 요소.
    - :nth-child(n): n번째 요소.
        - :nth-child(2n+1): 홀수 번째 (n=0부터 시작).
        - :nth-child(2n): 짝수 번째.
        - :nth-last-child(-n+3): 뒤에서부터 3번째까지(n+3은 3, 2, 1을 의미).
- 부정 선택자 (:not):
:not(.web)처럼 특정 선택자를 제외한 나머지를 선택.

---

4. 가상 요소 (Pseudo-Elements)

실제 HTML에는 없지만, 스타일을 적용하기 위해 가상으로 생성되는 요소.

- ::first-line: 문단의 첫 번째 줄.
- ::first-letter: 문단의 첫 번째 글자.
- ::selection: 사용자가 마우스로 드래그(선택)한 영역.
- ::before / ::after: 요소의 내용 *앞*이나 *뒤*에 가상의 요소를 만든다. 11_문자링크부정선택자.html ex. 파일에서 링크(a) 뒤에 href 속성값을 content로 붙여넣는 a:after { content : ' - ' attr(href); }

---

💡 (추가) 가장 중요한 개념: CSS 우선순위 (Specificity)

/* more specific and more direct 선택자가 더 우선순위를 갖는다. */

선택자 공부에서 가장 중요한 개념이 이 우선순위(Specificity). 여러 규칙이 동일한 요소를 선택할 때, 어떤 스타일을 적용할지 결정하는 '점수'.

점수가 높은 순서대로 적용:

1. !important ~~(가급적 사용 안 함)~~
2. 인라인 스타일 (HTML 태그에 style="..."로 직접 입력)
3. ID 선택자 (#header)
4. Class (.center), 속성 ([type]), 가상 선택자 (:hover)
5. 태그 선택자 (div), 가상 요소 (::before)
6. 전체 선택자 () (점수 없음)

예를 들어,
#nav > ul > li:first-child (ID 1개 + 태그 3개 + 가상 클래스 1개)는 :nth-child(2n+1) (가상 클래스 1개)보다 점수가 훨씬 높다.

그래서 10_구조선택자(가로메뉴).html 파일에서 홀수 번째(nth-child(2n+1))가 orange색임에도 불구하고, 더 구체적(specific)인 :first-child가 lightgray로 덮어쓸 수 있었던 것.
