5장 함수

1절 함수의 정의 및 사용 
	def 함수명(매개변수,...):
	- 오버로딩(같은 이름의 함수를 매개변수의 경우에 따라 달리 사용) 불가
	- 독스트링(docstring), 함수 정의 맨 윗줄에 설명 덧붙여 용이성 높임
	- 전역변수global vs 지역변수local (렉시컬lexical 특성)

2절 반환값이 필요하면 return 사용
	return a, b → (a,b) 튜플을 return

3절 함수의 매개변수
	매개변수가 기본값을 가질 때: 함수 호출 시 선택적으로 명시
	- 매개변수의 기본값이 스칼라 변수: 기본값은 함수 정의 시점
	- 매개변수의 기본값이 데이터구조 객체: 기본값은 함수 호출 시점

	키워드 인수: 매개변수명을 기입해 호출하는 경우 (sep='\t')
	튜플 매개변수: *args 가변 인수를 구현할 때 사용
	딕셔너리 매개변수: **kwargs 여러 개의 키워드 인수를 딕셔너리로 받을 때 사용
	언패킹: 
	- 튜플 매개변수 언패킹(*)
	- 딕셔너리 매개변수 언패킹(**)
	- 함수 정의 시 매개변수 순서 주의
		순서 인수 → 튜플 인수 → 키워드 인수 → 딕셔너리 인수
