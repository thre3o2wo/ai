Ch. 3 비지도학습 - 군집화

K-Means 알고리즘의 동작 원리를 파이썬 코드로 직접 구현해보고(Low-level), 이후 Scikit-Learn 라이브러리를 사용하여 효율적으로 처리하는 방법(High-level)과 최적의 군집 수(K)를 찾는 방법까지

---

1. 데이터 생성 (Data Generation)

군집화 실습을 위한 남/여 키, 몸무게 가상 데이터 생성.

- 데이터 구조: random.randint를 활용해 여성(40~75kg, 145~175cm)과 남성(60~95kg, 160~190cm) 데이터 각각 50개씩 무작위 생성 후 data 리스트 병합.
- 시각화: matplotlib.pyplot을 이용해 여성(빨강)과 남성(파랑) 데이터를 산점도로 시각화하여 분포 확인.

---

2. K-Means 로직 수동 구현 (Manual Implementation)

라이브러리 미사용, 알고리즘 핵심 원리 단계별 코딩.

- 초기화: 임의의 좌표 2개를 초기 기준점(Centroid)으로 설정 및 시각화.
- 거리 계산: 피타고라스 정리를 이용한 유클리드 거리 함수 dist(a, b) 정의.
    - 코드 구현: np.sqrt((a[0]-b[0])2 + (a[1]-b[1])2).
- 그룹 분류: 각 데이터와 두 기준점 사이의 거리를 비교, 더 가까운 그룹(group0 또는 group1)으로 데이터 할당.
- 중심점 갱신: 할당된 각 그룹 데이터의 평균 좌표(Mean)를 계산하여 기준점 위치 이동.

---

3. 반복 학습 시각화 (Iterative Visualization)

반복문(for)을 통한 학습 과정 확인.

- 반복 수행: for i in range(1, 10) 구문으로 분류 및 중심점 이동 과정을 9회 반복.
- 수렴 확인: 반복 수행 시마다 기준점(points)이 실제 데이터 군집의 중심으로 점차 이동(수렴)하는 과정을 그래프로 확인.

---

4. Scikit-Learn 활용 (Using Library)

수동 구현 로직을 라이브러리로 간결하게 처리.

- 모델 생성: KMeans 클래스 활용.
    - 설정: n_clusters=2 (군집 2개), init='random' (랜덤 초기화), n_init=10.
- 학습: model.fit(data) 실행.
    - 데이터 학습을 통해 최적의 군집 중심 탐색.
- 결과 확인:
    - model.cluster_centers_: 학습 완료 후 최종 결정된 중심점 좌표 확인.
    - model.labels_: 각 데이터별 할당된 그룹(0 또는 1) 레이블 확인.
- 예측: model.predict() 실행.
    - 새로운 데이터(예: [[100, 150]]) 입력 시 소속 그룹 판별.

---

※ [참고] 중요 개념: 응집도 (Inertia)

(주의: 실습 코드 미포함, K-Means 주요 개념 참조용)

- Inertia (응집도): 군집화 품질 평가 지표. 각 데이터와 중심점 간 거리의 제곱 합.
- 의미: 값이 작을수록 군집 내 밀집도가 높음을 의미.
- Elbow Method: 최적의 군집 수(K) 탐색 기법. K 증가에 따른 Inertia 그래프의 꺾임(Elbow) 지점을 최적값으로 선정.
