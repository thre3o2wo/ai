NumPy 패키지의 함수와 속성
- 배열 속성
	dtype : 요소들의 type
		astype(타입) : 요소 타입 변경
	itemsize : 요소의 byte size
	ndim : 차원수(축수)
	size : 요소 개수
	shape : 배열의 구조(튜플)
- 배열 생성
	np.array(obj, dtype, copy=T/F)
	np.zeros(tup, dtype)
	np.ones(tup, dtype)
	np.full(tup, 기본값, dtype)
	np.empty(tup, dtype)
	np.random.random(tup) : 0~1 균등분포 랜덤값
	np.random.uniform(low=0, high=1.0, size=tup) : low~high 균등분포 랜덤값
	np.random.normal(loc=0.0, scale=1.0, size=tup) : 평균loc, 표준편차scale 정규분포 랜덤값
	np.random.rand(n) : 균등분포의 1차원 난수 배열(난수 n개)
	np.random.randn(n) : 표준정규분포의 1차원 난수 배열(난수 n개)
	np.arange(from, to, by, dtype) : from부터 to 앞까지 by씩 건너뛴 1차원 배열 생성
	np.linspace(from, to, num) : from부터 to까지 포함하여 num개 숫자들을 1차원 배열로 생성
- 배열 차원 변경
	reshape() : -1 사용하면 자동계산
	resize() : -1 qnfrk
	T, transpose(): 전치행렬
	ravel(), flatten() : 1차원 변경
- 배열 연산
	요소별 연산, 행렬곱(@, 단위행렬 np.eye(size)), 집계함수(전체/축별, sum, min, max, mean, var, std)
	범용함수(np.add, np.multiply, ...), 브로드캐스팅
- 배열 합치기
	np.vstack(tup) : 수평
	np.hstack(tup) : 수직
	np.dstack(tup) : 열끼리 (짝지어) 합침
	np.column_stack(tup) : 1차원 배열을 열 단위로 2차원 배열로 합침
	np.row_stack(tup) : 행 단위로
	np.stack(tup, axis)
	np.r_[a, b, c] : row 단위로 쌓음 (한 행으로)
	np.c_[a, b, c] : column으로 쌓아서 합침 (1차원을 2차원으로)
	1차원배열[:, np.newaxis] == 1차원배열.reshape(-1,1) == np.expand_dims(a, axis=1)
		: 1차원 배열을 2차원 열벡터로 변환
- 배열 분리하기
	np.vsplit(ary, indices_or_sections) : 첫 번째 축 따라 분할
	np.hsplit(ary, indices_or_sections) : 두 번째 축 따라 분할
	np.dsplit(ary, indices_or_sections) : 3차원 이상에서 3번째 차원을 나눔
	np.split(ary, indices_or_sections, axis=?) : axis=0 == vsplit / axis=1 == hsplit / axis=2 == dsplit
	np.array_split(ary, indices_or_sections, axis=?)  : 균일히 나눠지지 않는 정수로도 split 가능
