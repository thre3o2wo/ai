@@ 가상환경 만들기 (ML-DL-NLP)

가상환경 목록 확인:
	conda env list
	(== conda info --envs)
가상환경 생성:
	conda create -n ml-dl-nlp python=3.10
	(-n : 가상환경 이름)
가상환경 활성화:
	conda activate ml-dl-nlp
pip 버전 최신화:
	python -m pip install --upgrade pip

ipykernel 설치
	conda install ipykernel
가상환경을 커널(jpykernel)로 등록
	python -m ipykernel install --user --name ml-dl-nlp --display-name "ml-dl-nlp (ipykernel)"
가상환경 커널 목록 확인
	jupyter kernelspec list

가상환경을 jupyter 커널 목록에서 삭제 하기 
	jupyter kernelspec remove 가상환경이름(ml-dl-nlp)
가상환경 지우기 
	conda env remove --name 가상환경이름

Java 17.0.17 설치
	환경변수 시스템변수  JAVA_HOME(C:\Program Files\Java\jdk-17) 등록
	Path변수에 아래의 폴더 추가
		%JAVA_HOME%\bin
		%JAVA_HOME%\bin\server
	
C++ (시스템에 설치돼 있지 않다면) v 2010 이상으로 설치
